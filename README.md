## Quick Start (Local Development)

```bash
# Clone your own repo
git clone https://github.com/KahlubDev/kahlubdev.github.io.git

cd kahlubdev.github.io/developerFolio-master

# Install dependencies
npm install

# Start development server
npm start
All content lives in one file:
src/portfolio.js

Edit src/portfolio.js (your name, bio, projects, links, etc.)
Save → preview with npm start
Commit & push:Bashgit add .
git commit -m "Update portfolio content"
git push
Deploy live changes:Bashnpm run deploy

Wait 1–5 minutes → refresh https://kahlubdev.github.io
